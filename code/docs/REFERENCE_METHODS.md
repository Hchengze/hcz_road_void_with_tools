## 2. Rayleigh 波绕射定位思想

二维文献常把空洞顶部或边界看作绕射点，用类似双曲线的走时曲线定位深度和水平位置。二维形式可写为：

```text
t(x) = (d + sqrt(x^2 + h^2)) / v
```

其中 `d` 是震源到绕射顶点的水平距离，`x` 是接收点相对绕射顶点的偏移，`h` 是深度，`v` 是有效 Rayleigh 波速度。

## 3. 三维改写

本项目不能只使用二维 `x-depth` 公式。三维候选点应为：

```text
p = (x, y, depth)
```

第一阶段采用保守的三维运动学走时：

```text
tau(s, r, p) = |s - p| / v + |r - p| / v
```

这里 `s` 是三维震源，`r` 是三维接收点或 DAS 通道，`p` 是三维候选空洞。横向 `y` 必须进入距离计算。

## 4. 多炮多道能量聚焦

DAS 空洞成像文献强调多炮和密集通道叠加的重要性。本项目第一版采用三维 travel-time energy stack：

```text
J(p) = 对所有 source 和 receiver，在 tau(s, r, p) 附近提取能量并叠加
```

候选目标函数可逐步扩展为：

- 理论走时附近的包络能量；
- semblance/coherence；
- phase-weighted stack；
- 已知震源子波下的 matched filtering。

第一阶段只实现绝对振幅能量栈，并保留完整 `J[x, y, depth]` 体。

## 5. 广义 S 变换拾取

相关文献用广义 S 变换在时频域拾取弱绕射能量。对本项目的启发是：

- 先压制直达面波或强背景能量；
- 在时频域寻找弱绕射能量峰；
- 把拾取时间转换为候选点残差；
- 用多炮多道残差形成三维目标函数。

当前阶段暂不实现 S 变换拾取，但后续可把拾取时间作为 `travel_time_energy_stack` 的替代输入。

## 6. DAS/光纤观测约束

DAS 文献强调：

- DAS 测量沿光纤方向的轴向应变或应变率；
- 每个通道有局部切向量 `e`；
- 真实弹性场下观测量近似为 `e^T epsilon(u) e`；
- gauge length 会对位移或应变做有限长度平均；
- 光纤耦合、通道间距和 gauge length 会影响高频浅层波。

因此当前点接收器近似必须明确标注为代理模型，不能声称已经实现真实 DAS。

## 7. 三维空洞散射

三维表面波空洞散射文献可作为后续中间物理模型参考：把空洞表面离散为等效散射面元或散射点，计算入射波在空洞表面的等效载荷，再叠加散射响应。

这比单点运动学绕射更真实，但仍不是完整数值波动方程。第一阶段暂不实现。

## 8. 不确定性分析启发

文献显示定位不确定性主要来自：

- 绕射能量弱、直达波强；
- 深部或小尺度空洞信噪比低；
- 层状介质导致速度不确定；
- 单侧光纤和单侧震源导致横向 `y` 与深度 `depth` trade-off；
- DAS gauge length 和通道间距改变高频响应。

本项目应保留完整三维目标函数体，并报告 half-max 区域、峰值比和 `y-depth` 混淆指标。

## 9. 第一版可用定位算法

第一版算法路线：

1. 建立三维搜索网格 `search_x, search_y, search_depth`；
2. 对每个候选点计算所有 source/receiver 的理论绕射走时；
3. 在记录中提取理论走时附近的能量；
4. 生成三维目标函数体；
5. 输出 `best_xyz`、误差、切片和不确定性；
6. 明确标注为三维运动学绕射定位，不是完整弹性反演。
