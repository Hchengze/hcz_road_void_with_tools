# tools 目录说明

本目录用于在本地存放外部三维正演工具源码、参考实现或临时试验副本，例如 Devito、OpenSWPC、SPECFEM3D、SW4 等。

这些外部工具通常是独立开源项目，包含各自的许可证、构建系统、示例数据和历史文件。为避免许可证混淆、仓库体积膨胀和第三方源码误提交，本项目默认不把 `tools/` 下的原始源码提交到 GitHub。

每个外部工具在进入本项目主流程前，都应记录以下信息：

- 工具名称和来源链接；
- 许可证；
- 主要物理方程和数值方法；
- 是否支持三维；
- 输入输出数据结构；
- 是否适合道路地下空洞和 DAS/光纤观测场景；
- 推荐复用方式和风险。

具体审计结论见 `code/docs/TOOLS_INVENTORY.md`。当前阶段只保留本说明文件进入 Git 仓库，外部工具源码请由用户在本地自行保存。
